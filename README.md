## ct519 docker compose

## ระบบบริหารจัดการข้อมูลลูกค้าและการบันทึกการติดต่อ (Customer \& Contact Note Management System)



### Features หลัก

* ประโยชน์ของโปรแกรม
1. สำหรับบันทึกการติดต่อกับลูกค้า
2. สามารถบันทึกได้ไม่จำกัดลูกค้า
3. สามารถบันทึกได้ไม่จำักดจำนวนการบันทึก



### Installtion

1. assume ว่า ขณะนี้ท่านได้ทำการติดตั้ง docker  ลงบนเครื่องของ ท่าน ตามระบบปฏิบัติการ ที่ท่านใช้งานอยู่แล้ว
2. git clone ลงในเครื่องของท่าน
3. เมื่อfile ทั้งหมด อยู่ในเครื่องแล้ว
4. ให้ท่าน อยู่ที่ path เดียวกันกับ file ชื่อ docker-compose.yml
5. ใช้คำสั่ง docker compose up -d --build  ระบบจะทำการดึงโปรแกรม ต่างๆ ที่เกี่ยวข้อง เข้าสู่ระบบ
6. ใช้คำสั่ง docker ps เพื่อตรวจสอบว่า มี container ถูกสรา้งขึ้นหรือไม่
7. เปิด broswe tabที่ 1 :  สำหรับในส่วนของ pgadmin
url: =>   http://<ip address เครื่องของท่าน>:5055
8. เปิด broswe tabที่ 2 :  สำหรับในส่วนของ web application
url: =>   http://<ip address เครื่องของท่าน>:5000
